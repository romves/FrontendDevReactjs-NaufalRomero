## Prerequisites

- Node.js 18.17.0
- React.js 18.2.0
- React Router 6.17.0


## How to start project

Clone the project

```bash
  git clone https://github.com/romves/FrontendDevReactjs-NaufalRomero
```

Go to the project directory

```bash
  cd FrontendDevReactjs-NaufalRomero
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
